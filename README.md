----- Install ChromeOS on Hard Drive -----
click browse as guest
and press ctrl+alt+F1
commands:
root
lsblk -e7
sudo chromeos-install -dst /dev/disk-name(name of disk)
exit(crtl+alt+F2)
shutdown system
boot from disk
continue with the installation 
have fun!
